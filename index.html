Follow these steps to enable Google Sheets bidirectional sync (including cross-device passwords):

  1. Open your Google Sheet → Extensions → Apps Script
  2. Delete all existing code in the editor
  3. Copy the full script below and paste it into the editor
  4. Click Save project (💾 icon)
  5. Click Deploy → New deployment
  6. Set Type: Web app · Execute as: Me · Access: Anyone
  7. Click Deploy → Authorize when prompted → Copy the Web App URL
  8. Paste that URL in the Sync tab above and click Save
